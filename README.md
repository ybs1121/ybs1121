# 안녕하세요, 백엔드 개발자 양병석입니다.

Java와 Spring 기반으로 B2B SaaS 백엔드를 개발하고 있습니다.

실무에서는 전자계약, 근로소득 연말정산, 이용권 검증, 거래수집, 금융관리 등  
데이터 정합성과 상태 흐름이 중요한 기능을 개발·운영해왔습니다.

운영 중 발생한 MyBatis 1차 캐시 계산 오류, PostgreSQL 조회 성능 문제,  
Excel 생성 CPU 병목처럼 실제 서비스에서 마주한 문제를 분석하고 개선하는 데 관심이 있습니다.

이 GitHub에는 실무에서 고민했던 정합성, 동시성, 캐시, 비동기 처리 문제를  
개인 프로젝트와 학습 기록으로 정리하고 있습니다.
